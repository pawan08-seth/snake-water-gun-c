# Snake Water Gun Game in C 🐍💧🔫

A simple console-based Snake Water Gun game written in C.

This project demonstrates fundamental programming concepts such as conditional statements, loops, random number generation, and user input handling.

## 🎮 Game Rules

- Snake drinks Water → Snake wins
- Water damages Gun → Water wins
- Gun kills Snake → Gun wins
- Same choice → Draw

## 🚀 Features

- Random computer choice generation
- User input handling
- Score tracking
- Loop-based gameplay
- Clean console interaction

## 🛠 Concepts Used

- if-else statements
- switch-case
- loops
- rand() and srand()
- functions
- basic I/O operations

## ▶️ How to Run

1. Compile:
   gcc game.c -o game

2. Run:
   ./game
